# suricato-web

## Pré-requisitos

- Node.js 20+
- npm 10+

## Dependências principais

| Dependência      | Versão |
|---|---|
| React            | 19     |
| Vite             | 6      |
| Tailwind CSS     | 4      |
| React Router DOM | 7      |
| Axios            | —      |

## Como executar

```bash
cd frontend
npm install
npm run dev
```

A aplicação ficará disponível em `http://localhost:5173`.
